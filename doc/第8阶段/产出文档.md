## 2. 结果

### 三基线 test 对比（MAE，Mbps）

| 模型 | MAE | RMSE | peak MAE |
|---|---|---|---|
| **本文模型** | **11274** | 37135 | 96808 |
| HistAvg | 16908 | 35678 | 87540 |
| LSTM | 18796 | 35201 | 82583 |

本文模型 MAE 最低，比 LSTM 优 40%、比 HistAvg 优 33%。

### 跨策略泛化

Dijkstra 11331 / ECMP 11358 / Queue-aware 11303，方差极小，泛化稳定。
