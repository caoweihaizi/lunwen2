# 第 6 阶段产出文档：数据可信度验证（实验一）

## 文档定位

P6 是硬门控审计阶段，不生成新数据，只审计 P1–P5 产出。产出为审计报告 + 实验一报告。

---

## 1. 代码模块（`src/audit/`）

| 模块 | 审计项 |
|---|---|
| `conservation.py` | OD 守恒 / 链路守恒 / commodity_load 还原 |
| `causality.py` | 突发先路由 / 故障先标签 / 缺失不改真值 |
| `stats.py` | 跨策略对比 / 空间相关 / 负载-丢包 / 周期性 |
| `missing_repro.py` | 缺失统计 / 可复现性 |

入口：`src/p6_main.py`。

---

## 2. 产出

- `data/processed/audit/p6_report.json`：12 项审计结果（PASS/FAIL + 证据）
- `doc/第6阶段/实验一报告.md`：论文初稿级实验一报告
- `results/metrics/<hash>_0_P6.json`：审计指标记录

---

## 4. 给 P7 的接口

P7（预测模型）可信任使用 P1–P5 产出：
- 训练数据：P5 `observed/`（含缺失观测 + 真值）
- 拓扑：P2 `topo/`
- 需求：P3 `demand/`
- 真值标签：P4 `sim/` link_state

数据特性见实验一报告 §9。
