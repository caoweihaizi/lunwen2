# 第 2 阶段产出文档：星座拓扑与覆盖生成

## 文档定位

登记 P2 阶段**实际产出**的全部交付物与星座几何特性，供 P3/P4/P7/P10 使用。所有数值基于 2026-07-07 真实执行（seed=0，config_hash `c1354c3acaf1`）。

## 1. 代码模块（`src/topo/`）

| 模块 | 关键接口 | 用途 | 使用阶段 |
|---|---|---|---|
| `orbit.py` | `WalkerConstellation` | J2 解析圆球轨道，卫星位置 | P2–P10 |
| `topology.py` | `build_planned_topology(positions, cfg)` | 四邻居计划拓扑（2轨内+2跨轨）+ 链路距离/时延 | P2/P4/P7/P10 |

## 3. 星座几何特性（真实执行结果）

### 3.1 星座配置

| 项 | 值 |
|---|---|
| 配置 | Walker Delta **6/66/F=2**（Iridium 类，F=2 为 P2 决策） |
| 倾角 | 53°（Starlink 类低倾角） |
| 高度 | 550 km |
| 轨道周期 | 95.5 min |
| 物理模型 | 简化 J2 解析圆球轨道（仅 RAAN 漂移），非 sgp4 |

### 3.2 拓扑

| 项 | 值 |
|---|---|
| 时隙数 | 56966（与 P1 GÉANT resampled 对齐） |
| 每星邻居 | 2 轨内（同面±1）+ 2 跨轨（右面最近可见）= 4，后处理强制 ≤4 |
| 度数分布 | {2,3,4}（边界卫星少邻居） |
| 每时隙边数 | mean 104, min 103, max 108（随跨轨可见性小幅变化） |
| 链路距离 | 1810–4994 km（受 max_isl=5000km 约束） |
| 传播时延 | 6.04–16.66 ms（mean 12.05 ms，ISL 典型范围） |
| 链路容量 | 10 Gbps（P3 标定可调） |

### 3.3 覆盖（关键特性）

| 项 | 值 |
|---|---|
| 最小仰角 | **5°**（P2 决策：10° 仅 84% 覆盖，5° 达 96%） |
| 覆盖半角 | 18.5°（地心角），半径约 2050 km |
| ±53°带覆盖率 | 0.964–0.977（4 个采样时隙） |
| 人口覆盖 | ±53° 带含 97.64% 人口 → 5° 仰角下覆盖约 96%×97.64% ≈ **93.7% 人口** |
| 高纬无覆盖 | ±53° 以外（2.36% 人口）无服务，属 53° 倾角星座固有特性 |

**关键提示给 P3**：覆盖采用**按需计算**（不预存全时隙全像元，5.84TB 不可能）。P3 用 `assign_primary_sat(positions[t], lon, lat, cfg)` 实时算每时隙主服务星。约 2.36% 高纬人口无服务，P3 处理：标记为无服务或就近分配到最南/最北卫星（P3 决策）。

### 3.4 势函数

| 项 | 值 |
|---|---|
| 样本目的 | [0,11,22,33,44,55]（每面第 1 颗） |
| 最远跳数 | 9（66 星直径约 9 跳） |
| 用途 | P10 action masking；P10 训练时按需算全部目的 |

## 5. 给 P3/P4/P7/P10 的接口契约

1. **位置序列**：`io.load_npz('topo/positions.npz')['positions']` → (T,66,3)，时刻见 `times.pkl`；
2. **拓扑边表**：`np.load('topo/topo_series.npz', allow_pickle=True)` 的 `edge_lists[t]` → (E_t,2) int32，`edge_dists[t]`/`edge_delays[t]` → (E_t,) float32；
3. **覆盖**：`assign_primary_sat(positions[t], lon, lat, cfg)` → (K,) sat id + (K,) 仰角，P3/P4 按时隙调用；
4. **势函数**：`potential_shortest_hops(adj, dst)` / `potential_propagation_delay(prop_delays, dst)`，P10 按需调用；
5. **重建邻接矩阵**：P4/P7 从稀疏边表重建稠密 adj：`adj = np.zeros((66,66),bool); adj[edges[:,0],edges[:,1]]=True; adj|=adj.T`。

**P2 不交付**：OD 需求合成、总量标定（P3）、故障注入（P4/P5）、仿真（P4）、模型训练（P7/P10）。
